# System Commands

This extension allows you to search operating system commands. Press enter to invoke the system command.

![Example](example.png)

## Supported system commands:

### Windows

-   Shutdown
-   Restart
-   Sign out
-   Lock
-   Sleep
-   Hibernate

### macOS

-   Shutdown
-   Restart
-   Log out
-   Sleep
-   Lock
